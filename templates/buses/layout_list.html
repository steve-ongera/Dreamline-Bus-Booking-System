About Seat Layouts

Seat layouts define the seating arrangement for buses. Each layout includes:

  • Total number of seats and rows
  • Seat positions and classes (VIP, Business, Normal)
  • Door position and aisle configuration
  • Visual representation for passengers

Tip: Create different layouts for different bus types (Standard, Luxury, VIP, Sleeper).
